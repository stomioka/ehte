Assess heterogeneity in treatment effect in clinical trials¶

The Python package ehte is designed to assess the heterogeneity of treatment effects between study arms. To evaluate the presence and quantify the degree of treatment response variability, it calculates the variance in the differences in cumulative responses between the placebo and active treatment arms across various percentiles. It tests null hypotheses ($H_0$): suggesting homogeneity in treatment effect. An alternative hypothesis indicating heterogeneity in treatment effect.

eHTE_Estimator estimates the standard deviation of individual treatment effects (ITE) in two ways. 1. Uses actual patient data to determine percentiles. 2. It calculates percentiles at intervals from 3 to 97 by 2 (a total of 48 percentiles).

This tutorial was executed on an ml.m5.large instance with 2 vCPUs and 8 GiB, meeting the minimum computing specifications for ehte. A larger trial data would require more RAM.

1. Assess heterogeneity in data with no heterogeneity in treatment effect¶

let's create a dummy data with no treatment heterogeneity¶

Input data is in ADaM format. Required columns for ehte is TRT01P, TRT01PN, and CHG. TRT01P should have string treatment arms including 'Placebo'. 'Placebo' should be coded as 1. ehte checks data for the requirements before heterogeneity assessment.

np.random.seed(123)

num_samples = 300
#mean 21.6, std=16.87
placebo_data = np.random.normal(loc=-21.6, scale=16.87, size=num_samples) 
drug_shift = -8
drug_data = placebo_data + drug_shift

data = pd.DataFrame({
    'TRT01P': ['Placebo'] * num_samples + ['Drug'] * num_samples,
    'TRT01PN':[1] * num_samples + [2] * num_samples,
    'CHG': np.concatenate([placebo_data, drug_data])
})

assess heterogeneity in treatment effect¶

class ehte.eHTE_Estimator(indf, n_perms =10000)

An eHTE estimator.

indf: pandas.DataFrame

  • Input Pandas data frame with TRT01P, TRT01PN, and CHG

    • TRT01P: str, 'Placebo' is one of the values
    • TRT01PN: int, TRT01PN=1 when TRT01P='Placebo'
    • CHG: int or float, Outcome values

n_perms: int, default=10000

  • The number of permutations
